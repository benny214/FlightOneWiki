# Tuning Guide

## Basic tune steps

1. Set Gyro Filters
1. Set D-term Filters
1. Enable SIM mode
1. Enable Whisper
1. PIDs
    1. Raise P (2-3 points) and D (10), D value can be 3-20 times the P value, all depending on your filters
    1. Add I-term
1. Add SIM Boost
